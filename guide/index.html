This section describes how to use the features of Alfred-Workflow.

Tip

If you’re writing a workflow that uses data from the system (e.g. from files/the filesystem or via command-line programs called via subprocess), please read Encoded strings and Unicode, which describes how to handle data from sources other than Alfred-Workflow’s APIs.
