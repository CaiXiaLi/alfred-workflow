Workflows using Alfred-Workflow¶

Here are some workflows that are made with Alfred-Workflow. Have a poke around in their repos for inspiration.

Adding your own workflow to the list¶

If you’d like your own workflow added to the list, please see the corresponding section in the GitHub README.

  • Alfred Backblaze by XedMada. Pause and Start Backblaze online backups.
  • Alfred Dependency Bundler Demo (Python) by deanishe. Demonstration on how to use the Alfred Bundler in Python.
  • Alfred Keyword Help by pochemuto. Show and filter available keywords.
  • Alfred Soundboard by Steffen. A soundboard for alfred at your fingertips.
  • Alfred-Flixsearch by deanishe. What’s available on Netflix and in which countries.
  • alfredwl by nicke5012. Wunderlist integration for Alfred2. Allows you to add tasks, show your lists, and show incomplete tasks through Alfred.
  • AppScripts by deanishe. List, search and run/open AppleScripts for the active application.
  • Base Converter by ahalbert. Convert arbitrary bases(up to base 32) in Alfred 2 and copy them to the clipboard.
  • BeautifulRatio by yusuga. This workflow calculates the Golden ratio and Silver ratio.
  • Better IMDB search by frankspin. Search IMDB for movies and see results inside of Alfred.
  • BibQuery by hackademic. Search BibDesk from the comfort of your keyboard.
  • Blur by Tyler Eich. Set Alfred’s background blur radius.
  • Calendar by owenwater. Displays a monthly calendar with Alfred Workflow.
  • Code Case by dfay. Case Converter for Code.
  • Codebox by danielecook. Search codebox snippets.
  • Continuity Support by dmarshall. Enables calling and messaging via contacts or number input.
  • Convert by deanishe. Convert between different units. No Internet connection required.
  • Date Calculator by MuppetGate. A basic date calculator.
  • Digital Ocean status by frankspin. Control your Digital Ocean droplets.
  • Display Brightness by fniephaus. Adjust your display’s brightness with Alfred.
  • Dropbox Client for Alfred by fniephaus. Access multiple Dropbox accounts with Alfred.
  • Duden Search by deanishe. Search duden.de German dictionary (with auto-suggest).
  • Fabric for Alfred by fniephaus. Quickly execute Fabric tasks.
  • Fakeum by deanishe. Generate fake test data in Alfred.
  • Forvo by owenwater. A pronunciation workflow based on Forvo.com.
  • Fuzzy Folders by deanishe. Fuzzy search across folder subtrees.
  • Genymotion by yakiyama. Start emulator instantly.
  • Gist by danielecook. An alfred workflow for accessing github gists as snippets. Supports tags, stars, and private gists.
  • Git Repos by deanishe. Browse, search and open Git repositories from within Alfred.
  • Glosbe Translation by deanishe. Translate text using Glosbe.com.
  • Gmail Client for Alfred by fniephaus. Manage your Gmail inbox with Alfred.
  • Google Drive by azai91. Browse, search and open Google Drive files from within Alfred.
  • HackerNews for Alfred by fniephaus. Read Hacker News with Alfred.
  • HGNC Search by danielecook. Search for human genes.
  • Homebrew and Cask for Alfred by fniephaus. Easily control Homebrew and Cask with Alfred.
  • IME by owenwater. A Input method workflow based on Google Input Tools.
  • iOS Simulator by jfro. Workflow for finding simulator app data folders, erasing apps and more.
  • IPython Notebooks by nkeim. Search notebook titles on your IPython notebook server.
  • Jenkins by Amwam. Show and search through jobs on Jenkins.
  • Jira Task Manager by miguelpuyol. A Jira Task Manager for Alfred.
  • Julian Date calculator by Tam-Lin. Converts dates to/from Julian dates, as well as some date math.
  • KA Torrents by hackademic. Search and download torrents from kickass.so.
  • Laser SSH by paperElectron. Choose SSH connection from filterable list.
  • LastPass Vault Manager by bachya. A workflow to interact with a LastPass vault.
  • LibGen by hackademic. Search and Download pdfs and ebooks from Library Genesis.
  • MailTo by deanishe. Send mail to contacts and groups from your Address Book.
  • Movie and TV Show Search by tone. Search for movies and tv shows to find ratings from a few sites.
  • Myinstants by flipxfx. A workflow to search Myinstants.com.
  • Network Location by deanishe. List, filter and activate network locations from within Alfred.
  • Packal Workflow Search by deanishe. Search Packal.org from the comfort of Alfred.
  • Pandoctor by hackademic. An Alfred GUI for Pandoc.
  • Parsers by hackademic. Greek and Latin parsers.
  • pass by mwest. Provide a minimal wrapper over the pass password manager (passwordstore.org).
  • Percent Change by bkmontgomery. Easily do percentage calculations.
  • PERT Calculator by agileadam. Generates accurate time estimates based on optimistic, realistic, and pessimistic expectations.
  • PHPStorm project opener by hansdubois. PHPStorm project opener.
  • Pocket for Alfred by fniephaus. Manage your Pocket list with Alfred.
  • Product Hunt by loris. List Product Hunt today’s hunts.
  • ProductHunt by chiefy. Read ProductHunt in Alfred.
  • PWS History by hrbrmstr. Retrieve personal weather station history from Weather Underground.
  • Quick Stocks by paperElectron. Add some stock symbols for Alfred to check for you.
  • Ramda Docs by raine. Search Ramda documentation.
  • Rates by Kennedy Oliveira. Simple exchange rates for alfred.
  • Readability for Alfred by fniephaus. Manage your Readability list with Alfred.
  • Reddit by deanishe. Browse Reddit from Alfred.
  • Relative Dates by deanishe. Generate relative dates based on a simple input format.
  • Resolve URL by deanishe. Follows any HTTP redirects and returns the canonical URL. Also displays information about the primary host (hostname, IP address(es), aliases).
  • Rotten Search by yakiyama. Search movie from RottenTomatoes.com.
  • Search Omnifocus by rhyd. This is a workflow that performs free text searches on OmniFocus data.
  • Searchio! by deanishe. Auto-suggest search results from multiple search engines and languages.
  • Secure Password Generator by deanishe. Generate secure random passwords from Alfred. Uses /dev/urandom as source of entropy.
  • SEND by hackademic. Send documents to the cloud.
  • Seq-utilies by danielecook. Fetch complement, reverse complement, RNA, and protein sequences. Generate random DNA. Blast a sequence.
  • Simple Timer by Paul Eunjae Lee. A very simple timer.
  • Skimmer by hackademic. Actions for PDF viewer Skim.
  • slackfred by frankspin. Interact with the chat service Slack via Alfred (multi-org supported).
  • Snippets by hackademic. Simple, document-specific text snippets.
  • Splatoon by flipxfx. A workflow with Splatoon helpers (maps, wiki).
  • Spritzr by hackademic. An Alfred Speed-Reader.
  • Stack Overflow by Que3216. Get answers to simple questions like “python function syntax”, without having to open your web browser.
  • StackOverflow Search by deanishe. Search StackOverflow.com from Alfred.
  • Steam by tresni. Activate your Steam codes & launch steam games with a quick keystroke or keyword.
  • Sublime Text Projects by deanishe. View, filter and open your Sublime Text (2 and 3) project files.
  • Torrent by bfw. Search for torrents, choose among the results in Alfred and start the download in uTorrent.
  • Travis CI for Alfred by fniephaus. Quickly check build statuses on travis-ci.org.
  • UberTime by frankspin. Check estimated pick up time for Uber based on inputted address.
  • URL craft by takanabe. A workflow that transforms a url into new one that allows some formats such as “Github Flavored Markdown link” or “shorten url” and so on.
  • VagrantUP by m1keil. List and control Vagrant environments with Alfred2.
  • Viscosity VPN Manager by deanishe. Manage Viscosity VPN connections.
  • VM Control by fniephaus. Control your Parallels and Virtual Box virtual machines.
  • Wikify by hackademic. Your little Evernote Wiki-Helper.
  • Workon Virtualenv by johnnycakes79. Workflow to list and start python virtualenvs (assumes you and have virtualenv and virtualenvwrapper installed).
  • Wowhead by owenwater. An Alfred workflow that helps you search World of Warcraft® database provided by wowhead.com.
  • Wunderlist3.alfredworkflow by gnostic. A Wunderlist 3 API cloud-based alfred workflow.
  • Youdao Dict by WhyLiam. 使用有道翻译你想知道的单词和语句.
  • ZotQuery by hackademic. Search Zotero. From the Comfort of Your Keyboard.
